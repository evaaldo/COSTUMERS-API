<h1>Costumer's API</h1>

<h4>✦ Resumo</h4>
<p>Uma API criada com fins de estudos para exercitar a criação de endpoints com Dapper, manipulando um banco local SQL Server</p>

<h4>✦ Ferramentas</h4>
<ul>
	<li>.NET 6</li>
	<li>Dapper</li>
	<li>SQL</li>
</ul>

<h4>✦ Endpoints</h4>
<ul>
	<li>GET - URL/Costumers</li>
	<li>GET - URL/Costumers/Name</li>
	<li>POST - URL/Costumers</li>
	<li>DELETE - URL/Costumers/{cpf}</li>
	<li>PUT - URL/Costumers/{cpf}</li>
</ul>
